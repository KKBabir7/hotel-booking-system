# Bootstrap 5.3.3

This project uses Bootstrap 5.3.3 via CDN for optimal performance and easy updates.

## CDN Links Used

### CSS
```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">
```

### JavaScript
```html
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>
```

## Why CDN?

- ✅ Faster loading times
- ✅ Automatic caching
- ✅ Always up-to-date
- ✅ No local file management
- ✅ Better performance

## Download Bootstrap (Optional)

If you want to use local Bootstrap files instead of CDN:

1. Visit: https://getbootstrap.com/docs/5.3/getting-started/download/
2. Download Bootstrap 5.3.3
3. Extract files to this folder
4. Update index.html to use local files

## Current Setup

The project is currently using CDN links in `index.html`. No local Bootstrap files are required.
